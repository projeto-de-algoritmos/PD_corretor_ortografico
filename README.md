## Instalação

**Linguagem**: Kotlin e Javascript<br>
**Framework**: React e Spring Boot<br>

## Uso

Deve-se ter instalado o Docker

Comandos:
Na pasta raiz execute

```
docker compose up --build
```

Abra o navegador e acesse o endereço http://localhost:5173/
